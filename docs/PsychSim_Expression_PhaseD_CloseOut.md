make them; each was measured, and the numbers are small — the direction §18 says means it is working.

## The gate class earned its keep — one architecture, two opposite functions
`PMC-l` is cortical (glutamatergic), so `PMC-l → PAG-PANIC` comes out **excitatory** — a suppressor that
excites is not a suppressor. The honest route is through a local interneuron, so the convention **answered the
long-registered "does `PAG-PANIC` need a gate?" — yes**, not by argument but because the edge could not be
written any other way (`vlPFC → ITC` exactly). And:
> `CeA` (GABAergic) → `vlPAG-GABA` ⊣ vlPAG = **release**;  `PMC-l` (glutamatergic) → `PAG-PANIC-GABA` ⊣
> PAG-PANIC = **suppression**. One local-interneuron architecture; **who reaches it decides.**

## The three questions — measured as-found

### 1. ★ S22 — the modality dissociation SURVIVED the crosstalk (now a finding, not an artifact)
`M1-face` reaches **both** effectors, so the dissociation is no longer guaranteed by an absence. Under
emotional drive (no volition): **pain → face** (NuFac 0.42, cry flat), **separation → voice** (cry 0.22,
PAG-PANIC 0.54, face flat). The crosstalk route exists but is engaged only by a volitional pose command
(`pose → face 1.0 AND cry 0.88`). **It survived contact with a route that reaches both — a real finding.**

### 2. Expressive suppression — Gross's signature, isolated and clean
Driving the suppression hub `PMC-l` directly (isolating it from `vlPFC`'s reappraisal — see below):
- **visible voice DOWN 53%** (cry 0.215 → 0.101), `PAG-PANIC` crushed to 0.000;
- **felt intensity UNCHANGED** (CeA 0.491 → 0.491; aIns 0.149 → 0.149) — suppression hides the display, not
  the feeling;
- **the face masks, not silences** (correct — the face has no suppressor, S3 fork ruling).
- **the autonomic cost does NOT emerge** (SympOut 0.137 → 0.137). **Principle 1: a finding that does not
  emerge is a missing pathway** — the question is *what route carries the autonomic cost of effortful
  suppression?* (candidate: effort/cognitive-load → sympathetic drive; `SympOut` is the correct terminal). Its
  own grounding pass (S28); the cost is NOT claimed and NOT coded here.

### ★★ HEADLINE — a psychological theory emerged from the connectome (reappraisal vs suppression)
Gross's process model — **antecedent-focused (reappraisal) vs response-focused (suppression)** regulation —
**fell out of where two grounded routes land.** Nobody coded "reappraisal changes the emotion, suppression
changes only the display." It fell out of the fact that one route reaches the amygdala's inhibitory gate and
the other reaches the expression column's:
- **reappraisal** = `vlPFC → ITC ⊣ amygdala` reduces the **feeling** (driving `vlPFC` drops CeA 0.491 → 0.228);
- **expressive suppression** = `PMC-l → PAG-PANIC-GABA` reduces the **display**, feeling intact.

**The distinction *is* the anatomy** (S27). This is the strongest demonstration to date that the substrate is a
**representation, not a restatement** — what six levels of descent were for. Engaging the full executive goal
(`vlPFC`) does both; isolating pure suppression means driving `PMC-l`, the suppression hub.

### 3. ★ The dual-pathway dissociation — EMERGED (the psychopathy expression signature)
Throttle `AFFECTIVE_EMPATHY`: the **involuntary** facial route is blunted (NuFac 0.42 → 0.11), the
**volitional** route is spared (pose → NuFac 1.0). Involuntary down, volitional intact — the signature,
emergent, never built. **`vlPFC → PMC-l` was load-bearing**: `vlPFC` is not in `AFFECTIVE_EMPATHY`, so the
volitional route's drive is independent of affect — the only reason this is measurable rather than
unmeasurable by construction.

## S29 — the omission confirmed, and `M1-face`'s open afferent is a SOCKET
**The reason I first gave — "including `PMC-l → M1-face` masks voice-down" — was wrong**, and is recorded as
such: that is omitting real anatomy because it changes a result we like, the `BA→dACC`-to-"low" failure shape
one step removed. Surfaced, not acted on — hence the ruling. The omission stands on **three reasons that
survive the result going either way**:
1. **`PMC-l` is LUMPED** — it both *suppresses* (lateral premotor → the faciorespiratory column) and *plans
   production* (premotor → M1); one node, two functions, so driving the suppressor necessarily drives
   production. Real cortex separates them (you can hold your voice without speaking). **`PMC-l` is the fourth
   split candidate** (after `dPAG` dl/l, `dACC`, `NuFac` upper/lower).
2. **`NuAmb-vocal` is lumped and the *display* cannot cope** — cry and speech genuinely share the laryngeal
   motoneurons (anatomically correct), but the display reads the pool's activation, so it cannot tell a cry
   from a sentence. "Cry goes up" when `PMC-l` drives `M1-face` is the display **misreading volitional
   vocalisation as distress** — the `NuFac` upper/lower read-out class (in reality a cry and a sentence are
   distinguished *acoustically*, not by which motoneurons fire).
3. **★ `M1-face` being afferent-less is CORRECT — it is a socket, the first hook of the climb back.** Its
   afferent is a **learned display rule** ("smile at the boss", "don't cry in front of strangers"), which
   lives in the representational-memory-executive layer (§2.16) — **D3, the level we descended from, not
   built**. So "posing is a capacity with no spontaneous trigger" is exactly right: the model *can* pose and
   has no reason to, because the reason is learned one level up. Wiring a trigger now would be a **coded
   display rule** — forbidden. An unafferented node whose afferent is a named, registered dependency one level
   up is not a defect; it is a **socket**. **Resolution: `M1-face`'s afferent lands when the representational
   layer's display rules exist.** The expression system is now complete except for its learned trigger — and
   the trigger is the thing we descended to get.
- **`NuFac` upper/lower (S25) — the highest-value split for the thesis.** With one scalar, "visible display"
  and "muscle activity" are the same number, so Gross's display-down-while-muscle-up cannot be represented;
  "displays without feeling" is anatomically a lower-face-without-upper (Duchenne) signature. Queued behind
  `dPAG` dl/l and `dACC`, flagged as most valuable.
- **The subcortical involuntary route is absent (S26) — disclosed against the dual-pathway result.** We have
  only the cortical involuntary route (`BA → dACC → NuFac`); the limbic/extrapyramidal route (Parkinson's
  masked face) is unbuilt, so our "involuntary" is the cortical half of a two-part system.
- **Autonomic cost gap (S28); the gate-class two-function insight recorded in `PAG-PANIC-GABA`'s provenance.**
