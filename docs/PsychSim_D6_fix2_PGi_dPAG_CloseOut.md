`SC-Pv` fires under distress (0.533), but **`dPAG` is a hard-gated escape *output* column** (dPAG-GABA, the

## The second cited arm (auditory) is also closed as wired — grain diagnostic done
The paper's other exteroceptive arm is the **inferior colliculus + paralemniscal zone → PGi**. Diagnosed
before routing (as ruled), and it is doubly blocked:
- **`AUD-brainstem` LUMPS the IC** (`nodes = "cochlear nuclei → superior olive → inferior colliculus"`) — a
  grain issue, not a naming one.
- **The cry bypasses it**: Phase C wired `IN-AUD:voice → A1-belt` (straight to cortex), not through
  `AUD-brainstem`. So even `AUD-brainstem(IC) → PGi` would not carry the vocal distress as wired.

**Registered, not built (S38), pending an auditory-grain pass.**

## ★ The rule at its cleanest (S39)
Refused `SC-Pv → PGi` (convenient, function-grounded) on the grounds, not the outcome. The paper then **(a)
confirmed `SC-Pv` is not a PGi afferent** (the cited set is somatosensory/auditory/autonomic — no
visual/collicular input) **and (b) named a different, grounded edge** (`dPAG → PGi`). Had the convenient edge
been routed, an invented edge would have done the work of a real one, the problem would have *looked* solved,
and the real edge would never have been found. *"Don't route it there because it would work"* is the thing that
finds the real mechanism — same consequence, opposite epistemics (ask what the paper says, not what dissolves
the entailment). That the real edge turned out **gated** is itself the honest anatomy: a real gate, not a
convenient number.
